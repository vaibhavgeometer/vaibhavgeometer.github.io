# Vaibhav's Blog

[![View Live Site](https://img.shields.io/badge/Live%20Demo-View%20Site-blue.svg)]((https://vaibhavgeometer.github.io/))

This is a brief description of my project. It serves as my personal blog to share insights on web development and technology trends.

## Features

* Clean and minimalist design.
* Fully responsive layout.
* Powered by the Jekyll static site generator.

## Technologies

* HTML5
* CSS3 (with Sass)
* Ruby / Jekyll

## Getting Started (Local Setup)

1.  Clone the repository:
    ```bash
    git clone [https://github.com/yourusername/your-repo-name.git](https://github.com/yourusername/your-repo-name.git)
    ```
2.  Install Jekyll dependencies:
    ```bash
    bundle install
    ```
3.  Serve the site locally:
    ```bash
    bundle exec jekyll serve
    ```
