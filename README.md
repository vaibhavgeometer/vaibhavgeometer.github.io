# Vaibhav's Blog

Welcome to my personal website, hosted on GitHub Pages. This site serves as a minimalist landing page connecting to my GitHub profile.

## About

This website is designed to be a simple, aesthetic entry point to my digital presence.

## Features

- **Minimalist Design**: Clean, distraction-free interface.
- **Responsive**: Adapts to different screen sizes.
- **Dark Theme**: Premium dark mode aesthetic.

## Tech Stack

- **Frontend**: HTML5, Vanilla CSS3, Vanilla JavaScript.
- **Design**: Custom CSS with glassmorphism effects and 'Outfit' typography.
- **Hosting**: GitHub Pages.
